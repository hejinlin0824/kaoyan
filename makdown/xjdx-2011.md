## 二、简单计算题（每题6分，共60分）  


1. 已知 \(f_1(t) = u(t)\) ， \(f_2(t) = u(t - 2) + \delta (t)\) ，试求 \(f_1(t) * f_2(t)\) 。


2. 已知某连续信号 \(f(t)\) 的傅里叶变换为 \(F(j\omega) = \frac{1}{j\omega - \omega^2}\) ，按照抽样间隔 \(T=1\) 对其取样得到离散时间序列 \(f(k)\) ，求 \(f(k)\) 的 \(Z\) 变换。  


3. 已知信号 \(x(t)\) 带限1kHz，对 \(x(2t)\) 进行抽样，求奈奎斯特频率。  


4. 某连续时间系统的输入输出关系为 \(y(t) = \int_{-\infty}^{t-1} x(\tau) d\tau\) ，判断该系统的线性时不变性。  


5. 计算 \(\int_{-\infty}^{-\infty} e^t u(-t) d\tau\) 的值。  


6. 已知 \(f(t) = \frac{\sin 6t}{t}\) ，求其傅里叶变换。  


7. 已知 \(f(t) = \sum_{n=0}^{\infty} e^{-(t-3n)} u(t-3n)\) ，求其拉氏变换 \(F(s)\) 。

8. 如果某信号的最高频率 \(4\mathrm{kHz}\) ,对其进行抽样,每个抛样点的数值用 8 位二进制表示,问持续一秒钟的信号需要用多少位二进制表示?  


9. 已知 \(x(n) = n \cdot \left(\frac{1}{2}\right)^n u(n)\) ,试求 \(ZT[x(n)]\) 。  


10. 判断 \(x(n) = \cos (0.1\pi n + 1)\) 是否为周期信号?  


三、(15 分)已知某 LTI 系统在激励 \(e^t = \sin t u(t)\) 时,零状态响应为 \(y(t) = e^{-t} u(t)\) ,求该系统的冲激响应并画出波形图。


四、(20分)某LTI系统的输入为 \(x(t)\) ,输出为 \(y(t)\) 。微分方程为  


\[y^{\prime \prime}(t) + y^{\prime}(t) + y(t) = x(t)\]  


1. 确定系统的传输函数 \(H(s)\) ;  


2. 求冲激响应 \(h(t)\) ,并画出波形;  


3. 画出零、极点分布图,并判断系统稳定性;  


4. 画出系统的模拟框图。
